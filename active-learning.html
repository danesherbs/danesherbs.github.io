A quick and dirty introduction to active learning
2 September 2020

In a standard supervised learning setup we're given labelled data. Though often in real-world problems we're not given a dataset -- we have to create one ourselves. But creating datasets is time-consuming, especially when data needs to be manually labelled. Not all labels are equally useful, though. Ideally we collect the most informative data at minimal cost. That is, we prefer quality over quantity.

That's the idea behind active learning.

In active learning, the algorithm has control over which data point is labelled next. It requires far fewer data points to be labelled.
